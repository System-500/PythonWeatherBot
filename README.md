# Weather Bot

A simple Telegram bot for checking weather information, developed in Python.

## Technologies
* **Telegram API:** [pyTelegramBotAPI](https://github.com/eternnoir/pyTelegramBotAPI)
* **Weather API:** [pyOWM](https://github.com/csparpa/pyowm)

## Features
* **Real-time Weather:** Get temperature, wind speed, and detailed weather conditions for any city.
* **Interaction:** Built-in commands and state handling to guide users through the query process.
* **Multilingual Support:** Weather data is retrieved in Polish.
* **Custom Responses:** Personalized welcome messages and visual feedback using stickers.
* **Fun Mode:** Responds to specific user inputs ("Du hast").
